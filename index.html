Visual Creator

Visual Creator is an UI designer for .NET on Windows and GTK on Linux.

With Visual Creator, you can create forms and basic programs for GTK, and create GUI using Windows Forms. Visual Creator provides its own file format for interfaces, that can be used to create projects both in Linux and Windows: the .vcui files. With GTK, you can also use XML.

Linux

On Linux, Visual Creator is built on GTK 4, Adwaita and Python. Is available as DEB, Flatpak and Snap, so it can be used on any type of Linux distribution.

To install, go to Releases. If you want to install it as Snap (recommended), visit the link to Snapcraft, or directly use the Snap Store. If you want to install as Flatpak, download the .flatpak file and install it. Finally, if you want to install it as a traditional Debian application, download the .deb file. Please note that the .deb file works only on Debian-based distributions, as Ubuntu, Linux Mint, elementary OS, Zorin, etc.

Windows

On Windows, Tinkta is built on .NET Framework, Windows Forms and C#.

To install, go to Releases and download the latest .EXE file of the newer version. Execute the installer, and finally start using Tinkta.

Source code and contributing

As you can see, Visual Creator is open source. You can also download the source code and build it yourself; you can contribute to the project; or you can create a fork of the project, since its under the GNU GPL 3.0.
